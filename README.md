#  NumberPuzzle App

A fun and challenging **number-matching puzzle game** built using **React Native** and **Expo**.  
Players can swipe, merge, and match numbers to reach higher scores ‚Äî inspired by popular games like *Number Master* but with a custom matching mechanic.

---

##  Features

-  Engaging number puzzle gameplay  
-  Smooth animations and transitions  
-  Sound effects using **expo-av**  
-  Optimized for Android (APK build via EAS)  
-  Fast performance with React Native Reanimated  
-  Responsive and clean UI design  
-  Supports both development and production builds
